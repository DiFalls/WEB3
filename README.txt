🍕 PIZZERIA DEL GATITO – WEB 3.0
================================

Um projeto experimental de visualização 3D inspirado em um dos mini-games clássicos do Club Penguin,
recriado no contexto do Metaverso / Web 3.0.
A ideia é trazer de volta a nostalgia da infância com uma pizzaria interativa em ambiente tridimensional.

--------------------------------
✨ OBJETIVO
--------------------------------
- 🎮 Criar uma experiência imersiva em 3D que simula o mini-game de pizzaria do Club Penguin
- 🌐 Explorar conceitos de Web 3.0, metaverso e visualização interativa
- 🛠️ Utilizar Unity como motor gráfico, com materiais, texturas e modelos organizados para fácil expansão

--------------------------------
📂 ESTRUTURA DO PROJETO
--------------------------------
O projeto está organizado em pastas que representam os principais elementos da pizzaria:

- 🏠 Materials/Cenario/        -> Materiais e texturas do ambiente (paredes, piso, teto)
- ⚙️ Materials/Objetos/Esteira -> Modelos e materiais da esteira de produção de pizzas
- 🍅 Materials/Objetos/Molhos  -> Materiais relacionados aos molhos
- 🍕 Materials/Objetos/Pizzas  -> Materiais da massa e borda da pizza
- 🖼️ Materials/Objetos/QuadrosDeInspiracoes -> Referências visuais (Pizzatron)
- ⏰ Materials/Objetos/Relogio -> Modelos e texturas do relógio decorativo

Cada pasta contém arquivos .mat (materiais), .png ou .jpg (texturas) e .fbx (modelos 3D).

--------------------------------
🚀 COMO EXECUTAR
--------------------------------
1. 📥 Clone este repositório:
   git clone https://github.com/Difalls/WEB3

2. 🖥️ Abra o projeto no Unity (versão recomendada: 2021 ou superior)

3. 📂 Carregue a solução: WEB 3.0 (Pizzeria del Gatito).sln

4. 🔎 Explore os assets na pasta "Materials" e visualize a pizzaria em 3D

--------------------------------
🐧 INSPIRAÇÃO
--------------------------------
Este projeto é inspirado no mini-game Pizzatron 3000 do Club Penguin,
trazendo uma releitura moderna para o metaverso.
A proposta é unir nostalgia + tecnologia em uma experiência interativa.
